## Сводка изменений

Реализованы три задачи:

### Пункт 17: Кнопка "Импорт прайса" (Excel/PDF/CSV парсинг)

**Компонент:** `components/construction/PriceImportModal.tsx`

**Функционал:**
- Импорт позиций прайс-листа из файлов Excel (.xlsx, .xls) и CSV
- Drag & drop загрузка файлов
- Автоопределение типа файла по расширению
- Парсинг с эвристическим определением колонок (код, название, единица, цена)
- Предпросмотр перед импортом с разделением на валидные/невалидные записи
- Автоопределение типа позиции (material/equipment/labor) по ключевым словам
- Нормализация единиц измерения
- Пакетный импорт (batch insert) по 50 записей
- Скачивание шаблона Excel

**Интеграция:** Добавлена кнопка "Importuj" (зеленая) в панель прайс-листа в `PriceLists.tsx`

### Пункт 18: Импорт nakładów из кошториса в оферту

**Компонент:** `components/construction/EstimateToOfferImportModal.tsx`

**Функционал:**
- Древовидное отображение структуры кошториса (etapy → zadania → zasoby)
- Чекбоксы для выбора на уровне etapu, zadania или pojedynczego zasobu
- Режимы импорта: "wybrane" / "wszystko"
- Группировка по типу (robocizna/materiał/sprzęt)
- Подсчет суммарной стоимости по категориям
- Автоматический расчет цены продажи с наценкой 20%
- Сохранение связей (source_resource_id, source_task_id, source_stage_id)

**Интеграция:** 
- Добавлена кнопка "Import z kosztorysu" в `Offers.tsx` (появляется когда есть linkedEstimateId)
- Автоматическое определение linked_estimate_id при загрузке оферты

### Пункт 19: Страница вместо модалки для запросов подрядчиков

**Компонент:** `pages/construction/ContractorRequestPage.tsx`

**Функционал:**
- Полноценная страница (не модалка) для работы с запросами подрядчиков
- URL: `/construction/contractor-request/:requestId`
- Трехколоночный layout:
  - Левая: данные клиента и объекта
  - Средняя: форма отправки запроса (выбор контактов, сообщение, срок, вложения)
  - Правая: полученные оферты от подрядчиков
- Выбор контактов чекбоксами
- Загрузка вложений
- Отображение полученных оферт с возможностью accept/reject
- Скачивание файлов оферт

**Интеграция:**
- Добавлен роут в `App.tsx`
- Кнопка "Zapytanie do podwykonawców" в модалке "Przygotuj ofertę" в `Requests.tsx`
